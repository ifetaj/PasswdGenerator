# Password Generator

Un générateur de mots de passe moderne, sécurisé et extensible. 
Ce projet fait partie de mon parcours d'apprentissage du développement web, de la programmation et de la versionisation avec Git/GitHub.

## Objectifs du projet

1. Créer un générateur de mots de passe robuste
2. Appliquer des pratiques modernes en sécurité et développement
3. Structurer un projet comme dans une vraie entreprise tech
4. Développer mon portfolio GitHub avec un projet propre, documenté et professionnel

## Fonctionnalités prévues

1. Génération de mots de passe aléatoires avec des longueures personnalisées
2. Choix des types de caractères (majuscules, minuscules, symboles, chiffres)
3. Option d'exclusion de caractères ambigus
4. Génération multi-mots de passe
5. Interface simple (CLI ou Web selon la version)
6. Tests automatisés
7. Bonnes pratiques de sécurité
